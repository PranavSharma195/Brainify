  5.  python manage.py runserver
  6.  Open http://127.0.0.1:8000

STEP-BY-STEP
────────────

1. Install dependencies
   pip install -r requirements.txt

   Optional (for real AI model):
   pip install tensorflow          (Windows / Linux)
   pip install tensorflow-macos    (Mac Intel / Apple Silicon)

2. Configure .env
   Edit the .env file and fill in:
     GROQ_API_KEY    — free at https://console.groq.com
     SECRET_KEY      — change for production
     EMAIL_HOST_USER / EMAIL_HOST_PASSWORD  (optional Gmail)
     GOOGLE_CLIENT_ID / GOOGLE_CLIENT_SECRET (optional OAuth)

3. Database setup
   python manage.py migrate

4. Create admin account
   python manage.py createsuperuser

5. Run
   python manage.py runserver

6. Browse
   http://127.0.0.1:8000          ← Landing page
   http://127.0.0.1:8000/admin-panel/  ← Brainify admin
   http://127.0.0.1:8000/admin/   ← Django admin

GOOGLE LOGIN (optional)
────────────────────────
  1. https://console.cloud.google.com → Create OAuth 2.0 credentials
  2. Authorised redirect URI: http://127.0.0.1:8000/auth/google/callback/
  3. Add to .env: GOOGLE_CLIENT_ID and GOOGLE_CLIENT_SECRET

AI MODEL
─────────
  Without TensorFlow: uses demo simulation (still shows metrics + UI)
  With TensorFlow:
    1. Open training/train_brain_tumor.ipynb in VS Code or Jupyter
    2. Run all cells (downloads BraTS 2020, trains Attention U-Net)
    3. Model auto-saves to core/brain_tumor_unet.h5
    4. Restart server → real AI model activates
